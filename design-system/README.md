The platform uses these pre-built component libraries:

- **shadcn/ui**: Pre-styled Radix components
- **Tailwind CSS**: Utility-first styling
- **Framer Motion**: Animations
- **Lucide React**: Icons

## üìä Design Tokens

All design values are stored as CSS custom properties:
- Colors: `var(--primary)`, `var(--secondary)`, etc.
- Spacing: Based on rem units (1rem = 16px)
- Border radius: `var(--radius)` = 0.5rem
- Shadows: Multiple elevation levels
